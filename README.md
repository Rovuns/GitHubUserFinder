# GitHub User Finder

**Автор:** Павел Тепляков

---

## Описание программы

*"GitHub User Finder - это приложение для поиска пользователей GitHub с возможностью добавления в избранное и сохранения списка в файл JSON. Использует GitHub API и графический интерфейс на Tkinter."*
---

## Требование для запуска

Что нужно установить на компьютер:

- Python 3.7 или выше
- Библиотеки: `pip install requests`

---

### Как запустить

```bash
git clone https://github.com/Rovuns/GitHubUserFinder.git
cd GitHubUserFinder

python main.py
